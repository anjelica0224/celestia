## what's inside

- **react native + expo** - built for iOS & Android
- **nativewind** - tailwind but for mobile
- **NASA Image and Video Library API** - pulls real space imagery from NASA's image library
- **custom event data** - astronomy events for 2025

## run it

```bash
npm install
npx expo start
```
